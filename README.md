# Luminous Landing

A beautiful full-stack landing page with a lightweight Express backend and a gorgeous modern UI.

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start the app

```bash
npm start
```

3. Open http://localhost:3000

## Docker

Build and run:

```bash
docker build -t luminous-landing .
docker run -p 3000:3000 luminous-landing
```
